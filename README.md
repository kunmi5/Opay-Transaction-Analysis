# Opay-Transaction-Analysis
Interactive Excel dashboard analyzing Opay transactions, cashflow trends, and spending patterns.
## Overview
__This project is an interactive excel dashboard designed to analyze personal financial transactions from Opay. It provides clear insights into cash inflow, spending patterns and transaction behavior over time.__

## Objectives
* Track monthly debit and credit trends
* Analyze spending across categories
* Identify top transaction recipients
* Monitor overall financial activity

 ### Key Features
* Dynamic dashboard with slicers for category filtering
* Monthly trend analysis (Debit vs Credit)
* KPI indicators:
  * Total Debit
  * Total Credit
  * Total Transactions
* Top recipient analysis
* Expense distribution visualization

### Tools & Techniques
* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Transformation
* Dashboard Design
